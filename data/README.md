# Data Directory

This folder should contain the raw dataset used for analysis.

## Dataset Details
- **Filename**: `IPL.csv`
- **Source**: Indian Premier League (IPL) ball-by-ball dataset (2008-2025).
- **Kaggle URL**: [IPL Dataset (2008-2025)](https://www.kaggle.com/datasets/saurabhbadole/ipl-dataset-2008-2025)

## Instructions
1. Download the dataset from the Kaggle URL above.
2. Unzip the file and rename the CSV to `IPL.csv` if it is not already.
3. Place `IPL.csv` in this directory (`data/`) before executing the notebook.
